# Kids Games - Arithmetic Game Implementation Guide

## Project Overview
This is a React-based educational games platform for children, with games designed to help them learn basic concepts through interactive gameplay.

### Game Features

#### Difficulty Levels
- **Easy (קל)**: Addition only - numbers range 1-10, simple addition problems
- **Medium (בינוני)**: Subtraction only - larger numbers (5+), always results in positive answers
- **Hard (קשה)**: Mixed operations - randomly switches between addition and subtraction

#### Game Mechanics
- **10 rounds** per game session
- **Multiple choice** - 4 options to choose from (1 correct, 3 wrong)
- **Mistake tracking** - counts total mistakes throughout the game
- **Best score saving** - stores best score per difficulty level in localStorage
- **Visual feedback**:
  - Correct answers: Green highlight + success animation
  - Wrong answers: Red highlight + shake animation
  - Confetti celebration on game completion

#### User Flow
1. Select difficulty level
2. Play 10 rounds of arithmetic problems
3. View final score (total mistakes)
4. Play again or change level

### Key Implementation Details

#### State Management
```javascript
- difficulty: Current selected difficulty
- rounds: Array of generated problems for the session
- currentRound: Current round number (0-9)
- selected: Currently selected answer
- totalMistakes: Running count of mistakes
- showCorrect: Flag to show correct answer
- gameOver: Flag to show game over screen
- roundMistakes: Mistakes in current round
```

#### Problem Generation
- `generateRound(difficulty)` - Creates one arithmetic problem based on difficulty
- `generateAllRounds(difficulty)` - Pre-generates all 10 problems for consistency
- Wrong answers are generated by adding/subtracting offsets from the correct answer

#### Sound and Animation
- Uses `playSuccess()` hook for correct answers
- Uses `playCheer()` hook for game completion
- Uses confetti library for celebration effects
- CSS animations for visual feedback (correctPulse, wrongShake, slideDown)

### Styling Approach

- **Responsive design**: Scales for mobile and desktop screens
